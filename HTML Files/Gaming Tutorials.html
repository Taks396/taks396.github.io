Description

This page serves to show off the different things I have done in the different videogames I play. It won't contain anything that can't be explained simply, like Call of Duty, where the objective is just get in and shooty shooty, but instead will contain games that require some explanation, like how I build something for example.

For all of these, I will include relavent screenshots, possibly blueprints designed in Gimp, so that you can replicate what I am doing, or use it as inspiration to do something yourself. Think of this as the nerdiest cooking recipe catalog ever.

7 Days to Die

This section is dedicated to the structures I have built in the survival crafting zombie apocolypse game called 7 Days to Die. Each structure will include volumetric dimensions of the build and instructions on how to build it. The instructions will assume you are building in creative mode using steel blocks. If you are building in survival mode, this will be the final form of the structure and it is your job to get there.

The Blender

Tested Releases: Alpha 19 to Full 1.4

Preface:

This base has been tested in every release specified but is significantly less viable following the release of Alpha 21. Due to the restriction of concrete upgrades behind Forge Ahead Magazine 30/75 and what feels like a block damage buff to zombies, this base will struggle early game but will function fine past blood moon 2. This also assumes that the player hasn't gotten insanely unlucky while looting for these magazines.

Minimum Dimensions (lwh):

  • Horde Base: 29x29x11 blocks
  • Ladder Tube: 3x3xH blocks where H is the verticle distance between the horde base and the factory
  • Factory/Storage: 15x15x5 blocks

How to Build:

  • Horde Base:
    1. Replace the top layer of dirt with blocks of length and width above leaving a one block hole in the center for a ladder.
    2. Starting from one corner, create an 8x8 grid of blocks seperated by 3 blocks. These should be blocks with a corner cut off oriented such that the corner is faced downwards and towards the ladder port, like this.
    3. Using normal blocks, extend these pillars up 6 blocks to a height of 7 total.
    4. Orient bar blocks to be flat with the top of these pillars and fill in the space. Leave gaps in the center of each 2x2 set of pillars, like this.
    5. In the gaps, place upside down facing IRON hatches (you'll need 49 total). These will serve as throwable ports, mild cover from cop spit, and access ports for wiring.
    6. Using the same corner cut blocks as in step 2, add an extra block to each of the pillars on the inner 6x6, then extend them up 2 normal blocks to a height of 3 (10 total pillar height). In both cases, these cut off corner blocks will allow the player to reach, and thus upgrade, the block beneath them which would otherwise be inaccessable.
    7. Add a bar block ceiling in the same way as you did the floor. Only leave a gap in the center of this bar ceiling for a ladder capped with a steel hatch for roof access.
    8. For the walls, create a 2 high wall of normal facing bars, leaving a 1 wide gap in the center of the walls. Do not place bars on the corners or on the third block up as you need a different block type for these.
    9. Select the corner bars block and fill in the corners only two high like the walls. Using this same block fill in the third block above the normal bars such that one side of the L bars is part of the roof, like this.
